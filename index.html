Finding the Inverse of an Exponential Function

Portfolio-style summary presentation

Learning Outcomes

  • Understand exponential and inverse functions
  • Learn how to find the inverse using base 10 logarithms

Exponential Function

An exponential function is written as f(x) = a × bx. It models:

  • Growth if b > 1
  • Decay if 0 < b < 1

Inverse of a Function

The inverse of a function swaps inputs and outputs. If a function maps x → y, the inverse maps y → x.

Sample Problem

Given: f(x) = 18x + 5

  1. Isolate the exponential term: f(x) - 5 = 18x
  2. Change variables: x - 5 = 18y
  3. Apply log base 10: log(x - 5) = y × log(18)
  4. Solve for y: y = log(x - 5) / log(18)

Final Answer: f⁻¹(x) = log(x - 5) / log(18)
